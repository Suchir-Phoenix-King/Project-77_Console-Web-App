About this IDE: Visual Studio Code is an Integrated Development Environment (IDE) made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

How to open Console Screen on Google Chrome?: First right click and you will get an option that says inspect. Click on it. At the top there will be a thing that says "Elements" with a blue underline. Beside it there is an option called console. Click on it and you will get the console code.
